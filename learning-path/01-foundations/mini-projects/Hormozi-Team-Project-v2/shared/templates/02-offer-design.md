## Document Purpose
Este template disena la oferta completa usando la Value Equation de Hormozi (Dream Outcome, Likelihood, Time & Effort). El Documento de Oferta traduce los insights del Contexto y el Customer Map en una proposicion de alto valor con planes, garantias y narrativa de transformacion.

**Prerequisites**:
- Documento de Contexto: `deals/[NombreDeal]/00-context/context-document.md`
- Customer Map: `deals/[NombreDeal]/01-customer-map/customer-map.md`
**Output**: `deals/[NombreDeal]/02-offer/offer-design.md`
**Self-critique**: `deals/[NombreDeal]/scratch/offer-self-critique.md` (ver `shared/references/scoring-rubric.md`)

## Completion Checklist

Before marking this Offer Document complete, verify:

- [ ] ONE SENTENCE PITCH captures transformation, not features
- [ ] AVATAR DAY 1 connects directly to Context Document insights
- [ ] DREAM OUTCOME is specific to Day 90 timeframe
- [ ] OBSTÁCULOS map to Context Document problems
- [ ] STACK has clear layered logic (Foundation → Intelligence → Control)
- [ ] Each PLAN has distinct value proposition
- [ ] ESCALERA provides fair upgrade path
- [ ] GARANTÍA reduces risk significantly and is deliverable
- [ ] VISIÓN paints compelling long-term narrative
- [ ] FRASE DE CIERRE is memorable and reframes the purchase
- [ ] All elements align with Hormozi Value Equation: Dream Outcome / (Perceived Likelihood of Achievement × Time Delay × Effort & Sacrifice)
